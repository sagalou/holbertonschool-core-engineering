# Functions and Modules

This directory contains Python scripts focusing on functions and modularity.

## Files
- `islower.py` - checks if a character is lowercase using ASCII
- `uppercase.py` - prints a string in uppercase using ASCII
- `print_last_digit.py` - prints and returns the last digit of a number
- `pow.py` - returns a to the power of b
- `add_0.py` - contains the add function
- `add.py` - imports and uses the add function
