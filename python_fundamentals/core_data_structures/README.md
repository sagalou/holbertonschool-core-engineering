# Python - Core Data Structures

## Description
This project focuses on the fundamental built-in data structures in Python 3. It covers the manipulation and logic behind lists, tuples, sets, and dictionaries. The goal is to learn how to store, group, and query data efficiently while respecting strict formatting and memory management constraints.

## Learning Objectives
* Iterate over lists and matrices to produce formatted output.
* Access and modify list elements safely.
* Understand the difference between mutable (lists, dicts) and immutable (tuples) types.
* Use sets for unique element operations (intersections, differences).
* Work with dictionaries to handle key-value mappings.
* Adhere to PEP8 styling and avoid external module imports.

## Requirements
* **OS:** Ubuntu 20.04 LTS
* **Language:** Python 3.8.x
* **Style Guide:** PEP8 (pycodestyle 2.7.x)
* **Execution:** All files must be executable and start with `#!/usr/bin/env python3`.
